> 我应该按什么按钮，可以全部把我输入的文本框里的文字全部删掉

**旺财**

在终端里，常用的快捷键：

- `Ctrl + U` — 删除光标前的所有内容
- `Ctrl + K` — 删除光标后的所有内容
- `Ctrl + A` 然后 `Ctrl + K` — 先跳到行首，再删到行尾，等于全清

这几个是 zsh/bash 通用的行编辑快捷键，在 Claude Code 的输入框里也能用。试试 `Ctrl + U`，最快。

> **我**
> 我是苹果电脑

**旺财**

macOS 终端里一样能用：

- `Ctrl + U` — 清掉光标前所有文字
- `Ctrl + K` — 清掉光标后所有文字

注意是 `Ctrl` 键，不是 `Cmd` 键。macOS 终端里的行编辑走的是 Ctrl 系列快捷键。

直接试 `Ctrl + U`，一键清空。

Mac 键盘上有的，叫 `control` 键（有的键盘上标的是 `⌃`），在键盘左下角，`fn` 键旁边。
