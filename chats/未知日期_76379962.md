#### **Step 2：利润率精算（必做！）**

很多新手死在没算清楚账，我给你一个公式：

```
【示例：售价$25的轻小件】

收入端：
售价：$25

成本端：
- 采购成本：¥20（1688价格）
- 头程物流：¥8/件（500件海运拼箱）
- FBA配送费：$3.5（<1磅小件）
- 亚马逊佣金：$25 × 15% = $3.75
- 广告成本：$25 × 25% = $6.25（ACoS按25%算）
- 其他成本：¥2（包装、标签、损耗）

总成本：(20+8+2)÷7 + 3.5 + 3.75 + 6.25 = $17.8
毛利：$25 - $17.8 = $7.2
毛利率：28.8%
```

**判断标准**：
- 毛利率 <25% → **不要做**（抗风险能力为0）
- 毛利率 25-35% → 勉强可做（需要运营能力强）
- 毛利率 >35% → **好品**（有优化空间）

**你必须用这个公式算清楚再下单！**

退货数量：2922 × 15% = 438单

单次退货成本：
1. FBA退货处理费：$2.50/件
2. 商品成本损失：
   - 如果能重新销售：损失0
   - 如果不能（污损/破损）：成本价¥20÷7 = $2.86
   - 假设50%能重新销售，50%报废：$2.86 × 50% = $1.43

3. 往返FBA配送费：$4.52（已经付过一次）

单次退货总损失：$2.50 + $1.43 + $0 = $3.93
（不算配送费，因为亚马逊不会二次收费，但你已经白付了）

月退货损失：438 × $3.93 = $1,721
占销售额比例：$1,721 / $30,008 = 5.7%
```

**结论**：15%退货率会吃掉**5-7%的利润率**

如果这个产品利润率是41%，扣除退货损失后：

### **策略2：提供训练指南**

**做法**：

- 视频演示（可以放listing里）

Day 6+: Gradually increase walk time

💡 Tip: Most dogs adapt within 3-7 days. Be patient!
```

**预期效果**：降低"狗不适应"退货30% → 总退货率再降**2-3%**

**做法**：

**预期效果**：降低质量问题退货50% → 总退货率再降**1-2%**
